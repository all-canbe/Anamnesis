<p>设计系统是产品一致性的基石，包含设计规范、组件库和文档三大要素。</p>
<h2>Design Token</h2>
<p>Design Token 是设计决策的原子化表达，独立于平台实现。</p>
<h2>色彩体系</h2>
<ul><li>主色+辅助色各含 9-11 色阶</li><li>对比度 ≥ 4.5:1 (WCAG AA)</li><li>建立中性色系统</li></ul>
<h2>组件设计原则</h2>
<ol><li>组合优于配置</li><li>受控与非受控并存</li><li>可访问性优先</li><li>一致的 API 命名</li></ol>
