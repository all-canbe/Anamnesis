### 头像

| | 浅色 | 暗色 |
|---|---|---|
| 用户头像背景 | `#4f6ef7`（纯色） | `#4f6ef7`（纯色，保持） |
| AI 头像背景 | `linear-gradient(135deg, #6ee7b7, #4f6ef7)` | `linear-gradient(135deg, #6ee7b7, #4f6ef7)`（保持） |
| 头像文字颜色 | `#ffffff` | `#ffffff`（保持） |
| 头像尺寸 | 32px 圆形 | 32px 圆形（保持） |

头像在两个主题下保持不变，因为头像本身的纯色和渐变在两种背景下都清晰可辨。

### 气泡

#### 用户气泡（右侧）

| 属性 | 浅色 | 暗色 |
|---|---|---|
| 背景 | `#1a1a2e`（深色） | `#e8e8f0`（浅色，与暗色背景形成反转对比） |
| 文字颜色 | `#ffffff`（白色） | `#1a1a2e`（深色） |
| 圆角 | 16px，右下角 4px | 16px，右下角 4px |
| 最大宽度 | 240px | 240px |

设计思路：用户气泡在浅色主题下用深色背景+白字，在暗色主题下反转为浅色背景+深色字，保证在两种背景下都有足够对比度。

#### AI 气泡（左侧）

| 属性 | 浅色 | 暗色 |
|---|---|---|
| 背景 | `#f0f0f5`（浅灰） | `#2a2a40`（深灰紫） |
| 文字颜色 | `#333333` | `#d0d0e0` |
| 圆角 | 16px，左下角 4px | 16px，左下角 4px |
| 最大宽度 | 240px | 240px |

设计思路：AI 气泡在浅色主题下是微弱的浅灰底，在暗色主题下是略深于背景的深灰紫底，与侧边栏背景形成层次区分但不突兀。

#### 气泡通用属性（双主题相同）

- 字号：13px
- 行高：1.6
- 内边距：10px 14px
- 头像与气泡间距：8px
- 自动换行：word-break: break-word

## 五、CSS 实现建议

### 使用 CSS 自定义属性（变量）实现主题切换

推荐在 `:root` 和 `[data-theme="dark"]` 中分别定义所有颜色变量，组件内统一引用变量名，切换时只需修改根元素的 `data-theme` 属性即可。

关键变量命名建议：

- `--bg-page` — 页面背景
- `--bg-sidebar` — 侧边栏背景
- `--bg-chat` — 对话区背景
- `--bg-input` — 输入框背景
- `--bg-bubble-user` — 用户气泡背景
- `--bg-bubble-ai` — AI 气泡背景
- `--text-primary` — 主文字
- `--text-secondary` — 次要文字
- `--text-bubble-user` — 用户气泡内文字
- `--text-bubble-ai` — AI 气泡内文字
- `--text-placeholder` — 占位符
- `--text-disclaimer` — 免责声明
- `--border-color` — 分割线
- `--color-accent` — 主题强调色
- `--color-accent-shadow` — 强调色阴影
- `--icon-color` — 图标颜色
- `--icon-hover-bg` — 图标 hover 背景

### 切换逻辑示例

```css
:root {
  --bg-sidebar: #ffffff;
  --bg-bubble-user: #1a1a2e;
  --text-bubble-user: #ffffff;
  /* ... 其余浅色变量 */
}

[data-theme="dark"] {
  --bg-sidebar: #1e1e30;
  --bg-bubble-user: #e8e8f0;
  --text-bubble-user: #1a1a2e;
  /* ... 其余暗色变量 */
}
```

切换时只需一行 JS：

```js
document.documentElement.setAttribute('data-theme', 'dark');
```

## 六、对比度检查要点

- 用户气泡：浅色模式下深色底+白字对比度 > 12:1；暗色模式下浅色底+深字对比度 > 12:1
- AI 气泡：浅色模式 `#333` 在 `#f0f0f5` 上对比度约 9:1；暗色模式 `#d0d0e0` 在 `#2a2a40` 上对比度约 8:1
- 主文字和标题：两种模式下对比度均 > 10:1
- 占位符和免责声明：有意的低对比度，仅作为辅助信息，可接受 3:1-4:1
